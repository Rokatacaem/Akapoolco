Tecnologías Principales:
Next.js 15 con App Router y React 19
PostgreSQL con pool de conexiones optimizado
TailwindCSS con sistema de diseño customizado
José (JWT) + bcrypt para autenticación
Vercel Blob para almacenamiento de archivos
Vercel como plataforma de deployment

Arquitectura:
Server Components y Server Actions ('use server')
Middleware de autenticación con control de roles
Separación por dominios (admin, referee, player)
Pool de PostgreSQL con SSL condicional

Prácticas de Desarrollo:
Principios SOLID y DRY
Seguridad OWASP (prepared statements, bcrypt, HTTP-only cookies)
Nomenclatura: kebab-case (archivos), PascalCase (componentes), camelCase (funciones)
Mensajes en español, código en inglés
Mobile-first responsive design

Convenciones:
Dark mode por defecto (esquema azul/slate)
Sistema de colores HSL con variables CSS
Validación y sanitización en backend
Manejo de errores con try-catch con PowerShell cuando se trate de infraestructura.
